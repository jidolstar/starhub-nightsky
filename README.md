# 🌌 Starhub Nightsky

> **고성능 웹 기반 천체 렌더링 엔진 & 인터랙티브 플라네타리움**

> 아직 개발 중입니다.

Starhub Nightsky는 밤하늘의 경이로움을 웹 브라우저와 모바일 환경에서 부드럽게 구현한 오픈소스 천체 엔진입니다. 스텔라리움(Stellarium) 수준의 부드러운 사용자 경험을 지향합니다.

🔗 **라이브 데모**: [https://jidolstar.github.io/starhub-nightsky/](https://jidolstar.github.io/starhub-nightsky/)

## ✨ 주요 특징 (Key Features)

### 🌓 시각적 몰입감
- **부드러운 페이드 효과**: 별자리, 지면, 격자 등 모든 레이어가 `0.03/frame` 속도로 부드럽게 나타나고 사라집니다.
- **지면(Landscape) 렌더링**: 셰이더 기반의 절차적 지형과 실루엣을 통해 실제 지평선과 같은 느낌을 제공합니다.
- **광각 조망**: 185도의 확장된 시야각(FOV)을 통해 밤하늘 전체를 한눈에 볼 수 있습니다.
- **천구 배경**: 밤하늘 현실감을 위해 고해상도 은하수 사진을 적용했습니다.

### 📱 최적화된 모바일 경험
- **반응형 UI**: 모바일에서는 밤하늘 시야를 확보하기 위해 설정창이 **하단 플로팅 드로워(Bottom Sheet)** 형태로 변환됩니다.
- **핀치 줌(Pinch-to-Zoom)**: 두 손가락 제스처로 매끄럽게 확대/축소가 가능합니다.

### 🔭 정밀한 천체 계산
- **방위/적도 좌표계**: 실시간으로 변화하는 두 가지 좌표계 격자를 동시에 지원합니다.
- **별자리 데이터**: 별자리 선, 명칭, 경계선을 개별적으로 제어할 수 있습니다.
- **실시간 시간 여행**: 과거와 미래의 밤하늘을 시간/일/월 단위로 자유롭게 탐험할 수 있습니다.
