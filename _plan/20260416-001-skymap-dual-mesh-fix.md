# 20260416-001-별지도 배경 레이어 이중 매쉬 재설계

현재 별지도 배경이 선처럼 늘어지거나 왜곡되는 현상을 해결하기 위해 매쉬를 물리적으로 분리한다.

## 배경
- 단일 구체 매쉬에서 두 개의 텍스처를 셰이더 분기문으로 샘플링할 때 경계면에서 보간 에러 발동.
- 이를 해결하기 위해 적경 0-12h, 12-24h용 독립 반구 매쉬를 사용한다.

## 상세 변경 사항
1. **매쉬 분할**: `this.mesh0`, `this.mesh1` 생성.
2. **기오메트리 설정**: `SphereGeometry(radius, 60, 30, 0, PI)`와 `SphereGeometry(radius, 60, 30, PI, PI)` 사용.
3. **셰이더 단순화**: 픽셀별 RA/Dec 계산 제거, 표준 UV 샘플링 적용.
4. **회전 동기화**: `THREE.Group`을 사용하여 두 반구를 동시에 회전.

## 기대 효과
- 스트레칭 왜곡 완벽 제거.
- 표준 UV 매핑을 통한 별 위치 정렬 정확도 향상.
