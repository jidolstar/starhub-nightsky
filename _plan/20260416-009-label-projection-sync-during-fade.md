# 20260416-009-페이드 중인 라벨의 투영 동기화 수정

## 1. 개요
별자리 이름(라벨)이 지평선 아래로 사라지는 동안 화면에서 고정되어 멈추는 현상을 수정한다. 지면 아래로 내려가더라도 페이드 애니메이션이 완료될 때까지는 마우스/카메라 이동에 맞춰 좌푯값을 계속 갱신해주어야 한다.

## 2. 사용자 리뷰 필요 항목
> [!IMPORTANT]
> - 지면(Horizon) 아래로 내려가는 즉시 계산을 중단하던 방어 로직을 완화합니다.
> - 라벨의 현재 투명도가 0보다 크다면, 지면 방향과 상관없이 화면상의 위치를 지속적으로 업데이트합니다.

## 3. 주요 변경 사항

### [MODIFY] [ConstellationLayer.ts](file:///d:/starhub-nightsky/src/starhub-nightsky/layers/ConstellationLayer.ts)
- `updateLabels` 메서드 수정:
    - `horizonFade`가 0이더라도 `labelOpacityMap`의 값이 0보다 크면 좌표 투영 계산을 수행하도록 변경.
    - 사라지는 도중의 라벨이 카메라 회전에 맞춰 부드럽게 화면 밖으로 혹은 지면 아래로 이동하도록 보장.

## 4. 검증 계획
- 별자리 이름이 지평선 아래로 가라앉을 때, 완전히 사라지기 전까지 마우스 드래그를 따라 실시간으로 위치가 변하는지 확인.
- 페이드 인/아웃 도중의 위치 연속성 확인.
