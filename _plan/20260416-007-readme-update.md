# 20260416-007-README 및 문서 최신화

## 1. 개요
별자리 데이터 아키텍처 리팩토링, 메인 렌더링 루프 최적화, 그리고 페이드 애니메이션 시스템 도입 등 대규모 변경 사항을 README.md에 반영한다. 외부 개발자나 사용자가 프로젝트의 구조와 사용법을 쉽고 정확하게 파악할 수 있도록 돕는다.

## 2. 사용자 리뷰 필요 항목
> [!IMPORTANT]
> - 기존의 `DefaultStarCatalog` 등이 제거되고 레이어 중심의 지연 로딩 구조로 변경된 점을 명확히 기술합니다.
> - 새롭게 추가된 비주얼 효과(페이드, 스텔라리움 스타일 격자)를 기능 목록에 추가합니다.

## 3. 주요 변경 사항

### 3.1 아키텍처 섹션 업데이트
- **Data Layer**: CSV 기반 매니저 구조와 지연 로딩 작동 방식 설명.
- **Engine Layer**: `SkyRenderer`의 중앙 루프 제어 기능 상술.
- **UI/Overlay**: DOM 오버레이와 WebGL의 결합 노하우 기술.

### 3.2 기능(Features) 목록 고도화
- 부드러운 전환(Fade In/Out) 애니메이션 추가.
- 고품질 천체 격자(Azimuthal, Equatorial) 시스템 설명.
- 지평선/적도선 및 방위 표시 레이어 소개.
